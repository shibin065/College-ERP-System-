The College ERP System is a comprehensive academic administration platform developed using Django. The system is designed to automate and manage key institutional workflows efficiently.

Core Features

Role-based authentication (Admin / HOD, Staff, Student)

Student registration and profile management

Staff management and subject allocation

Course and academic session management

Attendance recording and tracking

Leave application and approval workflow

Feedback system for students and staff

Result management and performance tracking

Secure login and authentication system

Media handling for profile images and documents

Technology Stack

Backend: Django (Python)

Database: SQLite (Development)

Frontend: HTML, CSS, Bootstrap

Authentication: Django Authentication System

System Modules

Admin Dashboard

Staff Dashboard

Student Dashboard

This system is designed for scalability and can be extended with modules such as fee management, library management, notifications, and reporting analytics.
